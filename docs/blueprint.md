# **App Name**: PortraitPro AI

## Core Features:

- Single Image Upload: Users can easily upload a single front-facing image of a person to begin the generation process.
- AI Portrait Generation (Initial): Utilize Nano Banana's latest model to generate an initial batch of 10 varied portrait images, including different angles, and half-body/full-body shots, based on the uploaded photo. This feature will use a tool to control generation parameters.
- Generated Image Preview: Provide a dynamic gallery where users can preview all generated images as thumbnails or in a larger view.
- Image Selection and Library: Allow users to select their preferred generated images and add them to a personal 'library' for safekeeping and final selection.
- On-Demand Additional Generation: Offer a button to generate 5 more diverse images if the user needs more options, appending them to the existing set of generated images. This feature will use a tool to control generation parameters.
- Library Image Management: Enable users to delete any unsatisfactory images from their selected library at any time.
- Developer Notification: Implement an email notification system to inform developers when a user has completed their image selections, optionally including a custom message from the user.

## Style Guidelines:

- The visual design will employ a dark color scheme to emphasize the images and evoke a professional, creative tool aesthetic. The primary accent color, a vibrant Sky Blue (#5A9FEB), is used for interactive elements and highlights, suggesting precision and innovation. The main background will be a very dark, subtly desaturated violet (#18161C), providing a calm and focused environment, while a deeper violet (#392673) will serve as a secondary brand color for branding elements.
- The application will primarily use 'Inter', a grotesque-style sans-serif font. Its modern, machined, and neutral aesthetic is highly readable and suitable for both headlines and body text, aligning with a sophisticated digital tool experience.
- Utilize clean, vector-based icons with a minimalist style that complements the dark theme and modern aesthetic. Icons will clearly represent actions like upload, generate, select, delete, and notification.
- Image previews and selections will be displayed in a responsive grid layout, allowing for easy scanning and interaction across different screen sizes. Key interactive elements, such as upload and generation buttons, will be prominently placed.
- Incorporate subtle, smooth animations for image loading, selection feedback, and transitions between gallery views to enhance the user experience without being distracting.
